inputs

atributos mais comuns nos imputs

autocomplete

autocompletando com emails por exemplo

autofocus

serve para o navegador focar automaticamente nesse campo (só funciona com o primeiro que estiver na página)

disabled

desativa o campo de texto

readonly

impede que o usuário escreve algum valor, mas com aparencia diferente do desabled

value

serve para definir um valor padrão para o campo de texto

form

serve para fazer um link do input ao form

name

serve para poder recuperar o campo que está linkado com o form

required

torna o input obrigatório na hora de submeter o form

placeholder

mostra um valor definido para o campo quando ainda não está preenchido

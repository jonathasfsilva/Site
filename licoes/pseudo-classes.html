Pseudo-classes

é um tipo de elemento que irá selecionar um elemento que estiver em um estado específico

Exemplo: é o primeiro elemento dentro de uma caixa, ou, o elemento que está com o ponteiro do mouse em cima dele

Pseudo classes começa com dois pontos seguidos do nome da pseudo class :pseudo-class-name

Selecionando elementos com pseudo-class

  • :first-child
  • :nth-of-type
  • :nth-child

Exemplo 1:

ul li:first-child{
    font-weight: bold;
    color: wheat
}

Combinadores, pois eles trabalham para buscar e combinar seletores a fim de aplicar uma estilização

Descendent combinator

  • Identificado por um espaço entre os seletores
  • Busca um elemento dentro do outro
  • Exemplo:
body main h3 {
    background-color: white;
    color: red;
    font-family: monospace;
}

H3 estilizado

também funciona dentro de outros elementos como div

Child combinator

  • Identificado pelo sinal > entre dois seletores
  • Seleciona somento os elementos que é filho direto do pai
  • Elementos diretos depois do filho serão desconsiderados
main > div > ul > li
  • item específico selecionado e modificado
    • sub-item iginorado

Adjacent sibling combinator

  • Identificado pelo sinal de + entre dois seletores
  • Seleciona somento o irmão direito que é irmão direto na hierarquia
